# CasualBongos-Lane Tap

## How do I install these formulae?

`brew install casualbongos-lane/tap/<formula>`

Or `brew tap casualbongos-lane/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "casualbongos-lane/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
